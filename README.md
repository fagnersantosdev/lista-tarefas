# 🚀 To-Do List API (Node.js & Express)

Esta é uma API RESTful simples e funcional para gestão de tarefas (To-Do List), desenvolvida como projeto de portfólio para demonstrar conhecimentos fundamentais em **Node.js** e **Express.js**.

## 🛠️ Tecnologias Utilizadas

- **Node.js**: Ambiente de execução Javascript.
- **Express.js**: Framework web para criação de rotas e APIs.
- **Nodemon**: Ferramenta de desenvolvimento para reiniciar o servidor automaticamente.
- **Thunder Client / Insomnia**: Para testes de endpoints.

## 📋 Funcionalidades (CRUD)

A API permite as seguintes operações:

- [x] **Listar Tarefas**: Retorna todas as tarefas registadas.
- [x] **Criar Tarefa**: Adiciona uma nova tarefa ao sistema.
- [x] **Atualizar Tarefa**: Permite editar o título ou o estado de conclusão (ID específico).
- [x] **Remover Tarefa**: Elimina uma tarefa do sistema (ID específico).

1. **Clona o repositório:**
   ```bash

3. **Inicia o servidor em modo de desenvolvimento:**

O servidor ficará disponível em: http://localhost:3000
